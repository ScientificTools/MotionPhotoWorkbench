## Third-Party Components

This project depends on third-party tools that are not covered by the repository MIT license.

### FFmpeg

- Upstream project: https://ffmpeg.org/
- Purpose in this project: frame extraction and video export
- Official download page: https://ffmpeg.org/download.html
- Recommended Windows provider for users of this repository: https://www.gyan.dev/ffmpeg/builds/

FFmpeg is required at runtime but is not redistributed in this repository.

Users are responsible for obtaining FFmpeg separately, reviewing its license terms, and placing `ffmpeg.exe` next to the application executable.
