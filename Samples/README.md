# Samples

This directory contains public sample assets for trying the application quickly.

Included files:

- `PXL_20260308_075633197.MACRO_FOCUS.RAW-01.MP.COVER.jpg`: source motion photo sample
- `animation.gif`: exported animated GIF
- `animation.mp4`: exported MP4
- `animation.webm`: exported WebM
- `animation.webp`: exported animated WebP
- `animationWeb.html`: small browser demo page for comparing exports

The sample `project.json` is intentionally not published because the original file contained local absolute paths tied to a personal machine.

## Privacy Note

The source sample is an actual motion photo. For this kind of file, metadata cleanup can interfere with the embedded video structure, so the sample is kept in a form suitable for testing the application. If you redistribute or fork these assets further, review them with your own privacy standards in mind.
